Online Examination System
Project Overview

The Online Examination System is a web-based application designed to conduct online exams efficiently. It provides an interface for students to take exams and view their results.

1. Features
- User-friendly interface
- Multiple-choice question support
- Timer-based examination
- Automatic score calculation
- Result display after exam submission
- Responsive design
  
2. Technologies Used
 HTML5
 CSS3
 JavaScript
